# Student Performance Prediction System

This is an end-to-end machine learning project built using Flask.

## Features
- Data preprocessing
- Machine learning model training
- Student score prediction
- Flask web application
- Deployment on Render

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML/CSS
